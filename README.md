# Anime MusicGen Fine-Tuning Project

This project fine-tunes Meta's MusicGen model specifically on Anime Openings to generate high-energy J-Rock and J-Pop style instrumentals.

## Setup Instructions
1. **Dataset**: The required audio dataset and `metadata.jsonl` are hosted on Google Drive. 
2. **Environment**: This project is designed to run in Google Colab with a T4 GPU.
3. **Installation**:
   Install the dependencies using:
   ```bash
   pip install -r requirements.txt

Note: After installing, you must restart the runtime in Colab to apply the library patches.
